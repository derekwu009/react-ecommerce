# Online Bookstore (WIP)

A full-stack e-commerce web application built with React, Express.js, and MariaDB.
This project demonstrates RESTful API design, JWT-based authentication, and persistent client-side state management.

## Overview

Users can:

- Browse available books
- Add/remove items from a shopping cart
- Cart state persists across sessions

The backend server and database currently run locally while the project is under active development.

## Tech Stack

### Frontend

- React
- CSS

### Backend

- Node.js
- Express.js

### Database

- MariaDB

### Authentication

- JSON Web Tokens (JWT)
- bcrypt for password hashing

### External Services

- Open Library Covers API for book covers
- placeholder.co (fallback placeholders for missing covers)
